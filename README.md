# Task Manager

Система для керування проєктами та задачами.

## Функціонал

- реєстрація за запрошенням, вхід
- створення проєктів
- створення задач
- зміна статусу задач
- канбан дошка
- коментарі
- сповіщення

## Технології

- React
- TypeScript
- ASP.NET Core
- MSSQL
- GraphQL

## Запуск

Команди для запуску будуть додані після налаштування проєкту.

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
dotnet restore
dotnet run
```
